# Função Definida por Partes

Uma função definida por partes é uma função que possui diferentes regras para diferentes intervalos.

**Exemplo:**

$$
\begin{cases}
\begin{gather*}
- 2x, \text{ se } x< 0\\
x^2, \text{ se } x \geq 0
\end{gather*}
\end{cases}
$$
