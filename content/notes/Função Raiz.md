# Função Raiz

Uma [[notes/Função|função]] raiz é definida por

$$f(x) = \sqrt[n]{x}$$

$$n \in \mathbb{Z}, n > 1$$

## Propriedades

- Domínio:
$$
\begin{cases}
\begin{gather*}
[0, \infty), \text{ Se } n \text{ é par} \\
\mathbb{R}, \text{ Se } n \text{ é ímpar} \\
\end{gather*}
\end{cases}
$$

- Imagem:
$$
\begin{cases}
\begin{gather*}
[0, \infty), \text{ Se } n \text{ é par} \\
\mathbb{R}, \text{ Se } n \text{ é ímpar} \\
\end{gather*}
\end{cases}
$$

- Raíz: $x = 0$
- Crescente em $\mathbb{R}$
- Paridade: Se $n$ é par, $f$ é ímpar.
